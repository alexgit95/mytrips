Résolution géographique mode

Mode actif : –

BigDataCloud est actif — résolution précise depuis les coordonnées GPS.
Mode hors-ligne — bounding boxes + centroïdes. Valeur par défaut depuis la variable GEO_API_ENABLED.
Ce changement est actif jusqu'au redémarrage. La valeur définitive se configure via GEO_API_ENABLED dans Portainer.
Géocodage inverse mode

Mode actif : –

Nominatim est actif — l'adresse sera récupérée depuis les coordonnées GPS pour la fonctionnalité "ici et maintenant".
Géocodage désactivé — les coordonnées GPS seront stockées directement. Utilise l'API Nominatim (OpenStreetMap) quand activé.
Ce changement est actif jusqu'au redémarrage. La valeur définitive se configure via GEOCODING_ENABLED dans Portainer.
